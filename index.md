# WaverPad - A simple soundboard made in PascalABC.NET

WaverPad is a very simple and lightweight soundboard program, made from scratch in PascalABC.NET by [Michael Agarkov](https://github.com/MichaelAgarkov).

![Screenshot_1](https://user-images.githubusercontent.com/81249219/139251754-690cccef-7090-42af-a4ae-0cab7bcce9e1.png)

You can read the [wiki](https://github.com/MichaelAgarkov/WaverPad/wiki) to find more info about this project.

## Download WaverPad

- [Releases](https://github.com/MichaelAgarkov/WaverPad/releases)
- [Source code](https://github.com/MichaelAgarkov/WaverPad)

### Compiler for the source code
I recommend using [PascalABC.NET version 3.4.2 build 2050](https://archive.org/download/pascalabc.net/PascalABCNETWithDotNetSetup.exe) because it's the version WaverPad is compiled in.
